[ Help Menu ]                    


Access each element by typing “/”followed by the respective number. 

E.g “/1” brings you to the “Add” section.


Type “/help” to access the Help Menu anytime.

  1. Add
  2. Delete
  3. Search
  4. Edit
  5. Mark Task as Done
  6. Undo / Redo
  7. Changing Location of Database
  8. Hotkeys and Functions

Type “/close” to close the Help Menu.
