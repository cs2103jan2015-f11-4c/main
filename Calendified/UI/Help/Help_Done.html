Marking a Task as Done

  • Mark a Task as Done                                                   
    • Type “done <index of the task>”


E.g. Typing “done 3” is marking the 3rd event displayed as done.


Note: You can only mark floating task as done. To delete Timed Tasks, please see the section on “Deleting a task” or simply type “/2”.
